# advanced-calender
A modern and visually expressive calendar web application designed to make scheduling more interactive, emotional, and creative.

Instead of traditional text-based planning, users can organize their workflow using emojis, images, colors, mood tracking, and draggable visual cards.

---

## Features

- Interactive monthly calendar
- Emoji-based scheduling
- Image uploads for daily plans
- Drag and drop task cards
- Mood tracking system
- Beautiful animated UI
- Responsive design
- Dark mode support
- Realtime syncing
- AI-powered planning assistance (future)

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase
- PostgreSQL

---

## Project Goal

The goal of this project is to transform traditional productivity tools into a more creative, visually engaging, and emotionally expressive experience.
